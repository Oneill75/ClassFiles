HTML-video-Element

Video mit Bedienelementen, selbststartend

Es soll möglich sein, die Video-Dateien small.mp4 bzw. small.ogv aus dem Ordner media abzuspielen, das Abspielen soll automatisch beginnen.
Bedienelemente sollen sichtbar sein.
Falls der benutzte Browser dazu nicht in der Lage ist, soll der Text "Das HTML-video-Element wird von diesem Browser nicht untersützt." ausgegeben werden.
Das Video soll 400 Pixel breit sein.
Nachdem Sie den Code zum Einbinden des Videos innerhalb der beiden HTML- Kommentare vollständig eingeben haben, zeichnen Sie diesen Absatz als "nicht mehr korrekt/zutreffend" aus.

Video ohne Bedienelemene, nicht selbststartend

Es sollen dieselben Dateien wie oben abgespielt werden können.
Bedienelemente sollen jedoch nicht sichtbar sein und das Abspielen nicht automatisch beginnen.
Falls der benutzte Browser dazu nicht in der Lage ist, soll der Text "Das HTML-video-Element wird von diesem Browser nicht untersützt." ausgegeben werden.
Das Video soll ebenfalls 400 Pixel breit sein.
Nachdem Sie den Code zum Einbinden des Videos innerhalb der beiden HTML- Kommentare vollständig eingeben haben, zeichnen Sie auch diesen Absatz als "nicht mehr korrekt/zutreffend" aus.

HTML-object-Element

PDF-Datei einbinden

Es soll möglich sein, die PDF-Datei abc.pdf aus dem Ordner docs wiederzugeben.
Das Objekt soll die gesamte Bidschirmbreite einnehmen. Falls der benutzte Browser das Objekt nicht darstellen kann, soll ein passender Text mit einem Link zu der PDF-Datei ausgegeben werden.
Nachdem Sie den Code zum Einbinden der PDF-Datei innerhalb der beiden HTML- Kommentare vollständig eingeben haben, zeichnen Sie diesen Absatz als "nicht mehr korrekt/zutreffend" aus.
